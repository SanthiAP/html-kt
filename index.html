HTML Basics

  • Split complex page into small simple containers
  • Use appropriate positions to place an element.
  • Use alt property in image tag (if image fails to load text will be visible instead image)
  • Prefer % or vh(View Height) or vw(View Width) unit to set height and width
  • Always try to use SVG images or i tags(font awesome, ...) instead PNG, jPEG (JPEG, PNG images will render the page slowly)
  • Avoid inline styles. eg: <div style="background: red;">
  • Try to set all css properties in class. Multiple classes can be used like <div class="class1 class2"></div>
  • Inline styles will overwrite class styles, class2 will overwrite class1's property
  • Try to set same font-family & font-size for entire project
  • Set uniform button height & width for entire project
  • Follow your application theme to pick button / font color
